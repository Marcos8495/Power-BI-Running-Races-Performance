# Análisis de Carreras de Running con BI
Proyecto de analítica de running donde centralizo mis entrenamientos en una base de datos y los exploto con dashboards en Power BI. Analizo ritmo, pulsaciones, altitud y rutas GPS para evaluar rendimiento, detectar patrones de esfuerzo y apoyar la mejora continua basada en datos.

## Objetivo del proyecto

Construir un sistema de Business Intelligence que permita:

Centralizar el historial de entrenamientos
Analizar rendimiento físico y evolución en el tiempo
Visualizar métricas clave de carrera de forma clara e interactiva

## Flujo de trabajo

**Recolección de datos**

Las actividades de running (distancia, ritmo, pulsaciones, altitud, recorrido GPS, etc.) se almacenaron en una base de datos estructurada.

**Modelado de datos**

Se realizó un proceso de limpieza, transformación y modelado para:

-Estandarizar métricas (ritmo min/km, pulsaciones medias, desnivel)

-Crear relaciones entre sesiones, parciales por kilómetro y métricas fisiológicas

-Habilitar análisis tanto por actividad individual como por agregados históricos

**Visualización en Microsoft Power BI**

Se desarrollaron dashboards interactivos que permiten explorar el rendimiento desde múltiples perspectivas.

## Principales indicadores analizados

**Rendimiento por distancia**

-Ritmo (min/km) a lo largo de la carrera

-Identificación de variaciones de intensidad y fatiga

-Comparación entre tramos estables y cambios de ritmo

**Respuesta fisiológica**

-Evolución de pulsaciones por distancia

-Relación entre frecuencia cardíaca y altitud

-Detección de zonas de esfuerzo

**Perfil de altitud**

-Cambios de elevación durante el recorrido

-Impacto del desnivel en el ritmo y las pulsaciones

**Análisis geoespacial**

-Visualización de rutas GPS sobre mapa

-Comparación de recorridos y tipos de terreno

**Métricas agregadas**

-Ritmo promedio por sesión

-Distribución de pulsaciones por zonas

-Ritmo por kilómetro (consistencia de pacing)

## Valor aportado

Este proyecto convierte datos deportivos en un sistema de soporte a decisiones personales, permitiendo:

-Evaluar mejoras en resistencia y velocidad

-Entender cómo responde el cuerpo ante distintos esfuerzos

-Optimizar estrategias de pacing

-Identificar patrones de fatiga o sobreesfuerzo

Además, demuestra competencias en:

-Modelado de datos

-Diseño de dashboards

-Storytelling con datos

-Aplicación de BI a datos del mundo real (IoT / wearables)

## Tecnologías utilizadas

-Base de datos relacional para almacenamiento histórico

-Procesos de transformación de datos

-Microsoft Power BI para modelado semántico y visualización

-Publicación y documentación del proyecto en GitHub

## Conclusión

Este proyecto muestra cómo la analítica de datos puede aplicarse al rendimiento deportivo individual, transformando entrenamientos cotidianos en insights cuantificables y visualmente interpretables. Es un ejemplo práctico de cómo combinar deporte, datos y BI para generar conocimiento útil y medible.
